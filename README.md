# NusaKarya Landing Page

Landing page/company profile **NusaKarya** (nusakarya.id) dibangun dengan Vue 3 + Vite + Tailwind CSS.

## Menjalankan project

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
npm run preview
```

## Struktur utama

- `src/components/layout` - Navbar & Footer
- `src/components/sections` - Semua section landing page
- `src/components/common` - Komponen kartu, tombol, FAQ
- `src/data` - Konten layanan, portfolio, paket, FAQ, site config
- `src/composables/useSeo.ts` - helper meta title/description
